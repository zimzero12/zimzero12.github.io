Welcome to EchoDraw!

EchoDraw is the easiest programming language ever, designed for kids! You can use it to print messages and draw cool shapes.

How to Use EchoDraw

  1. Open the index.html file in a web browser to use the block editor.
  2. Drag and drop blocks to build your program.
  3. As you add blocks, the text-based code will be generated (we will build this part next).
  4. Copy the text-based code and paste it inside the triple quotes in the run_echodraw.py file.
  5. Run the program from your Terminal to see your creation!

EchoDraw Commands

Here are all the commands in EchoDraw:

print [message]

  • What it does: Displays a message in the Terminal.
  • Example: print Hello there!

move_forward [pixels]

  • What it does: Moves the turtle forward by a certain number of pixels.
  • Example: move_forward 50

turn_right [degrees]

  • What it does: Turns the turtle to the right by a certain number of degrees.
  • Example: turn_right 90
