Welcome to EchoDraw!

This is the instruction manual for EchoDraw. It contains a guide for beginners and a full language specification for experts.

User Guide for Creators

This guide explains how to use the EchoDraw application.

The Blocks Reference

Events (New!)

The new Events category is the key to making interactive programs and games! Blocks in this category are special "hat" blocks. They don't connect on top of anything else; they are the start of their own script.

Use the when key pressed block to make your program listen to the keyboard. Any code you put under this hat will only run when you press that specific key. Try making an Etch A Sketch by moving the turtle 10 pixels for each arrow key!

My Blocks

Use the "My Blocks" category to create your own custom commands. This is great for organizing your code and reusing parts you build often, like a block to draw a house or a star.

Other Blocks

Use the **Drawing**, **Artist**, and **Logic** blocks to control the turtle, change its appearance, and create repeating patterns.

EchoDraw Language Specification

EchoDraw is a procedural, imperative, event-driven, turtle graphics language. The interpreter performs a pre-pass to register all function and event definitions before executing the main program logic.

1. State Control Commands

move_forward `distance`

  • Moves the turtle forward by an integer number of pixels.

turn_right `degrees`

  • Rotates the turtle clockwise by an integer number of degrees.

2. Pen Attribute Commands

pen_up / pen_down

  • Lifts or lowers the pen.

set_color `color`

  • Sets the line color (e.g., `set_color red`).

set_thickness `width`

  • Sets the line thickness in integer pixels.

3. Flow Control Commands

repeat `count` ... end_repeat

  • Executes the enclosed block of code a specified number of times.

4. Function Definition Commands

define `name` ... end_define

  • Defines a new command with the specified `name`.

call `name`

  • Executes a previously defined function.

5. Event Handling Commands (New)

on_key `key_name` ... end_on_key

  • Description: Defines an event handler. The enclosed block of code is not executed immediately. Instead, it is registered and will only execute when the corresponding keyboard key is pressed. The `Run Program` button must be clicked once to activate these listeners.
  • Argument: `key_name` (string). The specific key to listen for. Valid values are `ArrowUp`, `ArrowDown`, `ArrowLeft`, `ArrowRight`, and `Space`.
  • Example:
    on_key ArrowUp
      move_forward 10
    end_on_key

    on_key ArrowLeft
      turn_right -10
    end_on_key
