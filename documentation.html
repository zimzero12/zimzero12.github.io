Welcome to EchoDraw!

Hi! This is the instruction manual for EchoDraw, the programming language I made for you to learn and have fun with. It's now upgraded with an Artist's Toolkit!

The Blocks

These are the puzzle pieces you can use to give the computer instructions.

Drawing Blocks

move forward: Tells the turtle to move forward and draw a line.

turn right: Tells the turtle to turn in place.

The Artist's Blocks (New!)

set color to [red]: Changes the color of the pen. You can pick from the list!

pen [up/down]: Lifts the pen up so it doesn't draw, or puts it down to start drawing again. This is how you draw separate shapes!

set thickness to [5]: Changes how thick your drawing line is. Try a big number like 20!

The Logic Block

repeat: Any blocks you put inside this will happen again and again. It's the best way to make shapes easily.

Example Programs to Try

Challenge: Draw a Colorful House!

This challenge uses all the new tools. We'll draw a blue square for the house, and then a separate red triangle for the roof.

  1. First, draw the blue square:
  2. Use the set color to [blue] block.
  3. Use the set thickness to [10] block to make it chunky.
  4. Use a repeat 4 block to draw a square (move 100, turn 90).
  5. Next, get ready to draw the roof:
  6. Use the pen [up] block to stop drawing.
  7. Move to the top corner of the house (you can use move forward 100).
  8. Use the pen [down] block to start drawing again.
  9. Finally, draw the red roof:
  10. Use the set color to [red] block.
  11. Use a repeat 3 block to draw a triangle (move 100, turn 120).

Click "Run Program" and see your house appear!
