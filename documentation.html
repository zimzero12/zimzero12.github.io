User Guide for Creators

This guide explains how to use the EchoDraw application.

The Main Buttons

Run Program, Save Drawing, and Share Program: These buttons let you run your code, save your art as an image, and create a special link to share your program with anyone.

The Blocks Reference

Drawing & Artist Blocks

Use blocks like move forward, turn right, set color, and pen up/down to control the turtle and create your art.

Logic Blocks

The repeat block is great for drawing shapes that have repeating sides, like squares or stars.

My Blocks (New!)

This is the most powerful feature! In the "My Blocks" category, you can click "Make a Block" to invent your own commands. If you have a set of blocks you use all the time (like drawing a square), you can give it a name like "draw_square". A new draw_square block will appear that you can use anywhere, as many times as you want!

EchoDraw Language Specification

EchoDraw is a procedural, imperative, turtle graphics language. The interpreter performs a pre-pass to register all function definitions before executing the main program logic line-by-line.

1. State Control Commands

move_forward `distance`

  • Moves the turtle forward by an integer number of pixels.

turn_right `degrees`

  • Rotates the turtle clockwise by an integer number of degrees.

2. Pen Attribute Commands

pen_up

  • Lifts the pen; movement will not draw a line.

pen_down

  • Lowers the pen; movement will draw a line.

set_color `color`

  • Sets the line color (e.g., `set_color red`).

set_thickness `width`

  • Sets the line thickness in integer pixels (e.g., `set_thickness 5`).

3. Flow Control Commands

repeat `count` ... end_repeat

  • Description: Executes the enclosed block of code a specified number of times. Can be nested.
  • Example:
    repeat 4
      move_forward 50
    end_repeat

4. Function Definition & Control (New)

define `function_name` ... end_define

  • Description: Defines a new custom command. The `function_name` must be a single word (no spaces). All code between `define` and `end_define` is stored as the body of the function. These blocks are not executed immediately.
  • Argument: `function_name` (string).

call `function_name`

  • Description: Executes the body of a previously defined function. The interpreter will jump to the function's code, execute it, and then return to the next line in the main program.
  • Argument: `function_name` (string).
  • Example:
    define draw_square
      repeat 4
        move_forward 100
        turn_right 90
      end_repeat
    end_define

    call draw_square
