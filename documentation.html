Welcome to EchoDraw!

This is the instruction manual for EchoDraw. I've added some powerful new tools to help you save, share, and even become a pro coder!

The Main Buttons

You now have three powerful buttons to control your projects.

Run Program: Click this to run the code in the editor and see your turtle draw on the canvas.

Save Drawing: This downloads a picture (a PNG file) of your art to your computer. You can print it or use it as a background!

Share Program: This creates a special magic link. You can send this link to anyone, and when they open it, they'll see your code and your drawing!

How to Create

There are now two ways to make art in EchoDraw.

  1. The Visual Way: Drag and drop blocks into the editor. You'll see the text code appear in the Code Editor box.
  2. The Pro Way: Click inside the Code Editor and type your commands directly!

Becoming a Pro: The Code Editor

The Code Editor is your path to becoming an expert. The blocks are a great way to start, but programmers write code with text.

You can start a program with blocks, and then click into the editor to change the numbers or colors. Try building a square with blocks, then change move forward by 100 to move forward by 150 in the text and run it again. See how it changes? You're coding!

The Blocks Reference

Here are all the blocks you can use.

Drawing Blocks

move forward & turn right: The main blocks for moving and turning the turtle.

Artist's Blocks

set color, set thickness, and pen up/down: Use these to change how your lines look and to draw separate shapes.

Logic Blocks

repeat: The most powerful block for drawing shapes with repeating patterns, like squares or spirographs.
