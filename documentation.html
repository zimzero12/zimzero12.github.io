User Guide for Creators

This guide explains how to use the EchoDraw application.

The Main Buttons

Run Program: Executes the code in the editor and draws the result on the canvas.

Save Drawing: Downloads a PNG image of the canvas.

Share Program: Generates a unique URL that contains the current program, allowing anyone with the link to load, view, and run your code.

How to Create

  1. The Visual Way: Drag and drop blocks. The text code will be automatically generated in the Code Editor.

    EchoDraw Language Specification

    EchoDraw is a procedural, imperative, turtle graphics language. The interpreter processes commands line-by-line, from top to bottom. The language is case-sensitive and whitespace-sensitive for command arguments.

    1. State Control Commands

    These commands modify the state of the turtle (its position and angle).

    move_forward `distance`

    • Description: Moves the turtle forward from its current position in the direction it is facing. If the pen is down, it draws a line.
    • Argument: `distance` (integer). The number of pixels to move.
    • Example: move_forward 150

    turn_right `degrees`

    • Description: Rotates the turtle clockwise from its current orientation. Does not draw a line.
    • Argument: `degrees` (integer). The angle of rotation in degrees. Negative values produce a counter-clockwise turn.
    • Example: turn_right 90

    2. Pen Attribute Commands

    These commands control the properties of the drawing pen.

    pen_up

    • Description: Lifts the pen from the canvas. Subsequent `move_forward` commands will change the turtle's position without drawing a line.
    • Arguments: None.

    pen_down

    • Description: Lowers the pen onto the canvas. Subsequent `move_forward` commands will draw a line.
    • Arguments: None.

    set_color `color`

    • Description: Sets the stroke color for all subsequent lines. The initial color is `black`.
    • Argument: `color` (string). Accepts standard CSS color names (e.g., `red`, `blue`) without quotes.
    • Example: set_color orange

    set_thickness `width`

    • Description: Sets the stroke width for all subsequent lines. The initial thickness is `2`.
    • Argument: `width` (integer). The thickness of the line in pixels.
    • Example: set_thickness 10

    3. Flow Control Commands

    These commands control the order in which code is executed.

    repeat `count`

    • Description: Marks the beginning of a loop block. The lines of code between this command and its matching `end_repeat` will be executed a specified number of times. Loops can be nested.
    • Argument: `count` (integer). The number of times to execute the loop body.
    • Example: repeat 4

    end_repeat

    • Description: Marks the end of a `repeat` loop block. Must have a corresponding `repeat` command before it.
    • Arguments: None.
