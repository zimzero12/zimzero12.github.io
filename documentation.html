Welcome to EchoDraw!

Hi! This is the instruction manual for EchoDraw, the programming language I made for you to learn and have fun with.

How to Use EchoDraw

  1. On the left side of the screen, you'll see a list of code blocks.
  2. Drag the blocks you want into the big white space to build a program. They snap together like puzzle pieces!
  3. When you think your program is ready, click the big green "Run Program" button.
  4. Watch the right side of the screen to see a turtle draw what you programmed!

The Blocks

These are the puzzle pieces you can use to give the computer instructions.

The move forward Block

This tells the turtle to move forward and draw a line. You can change the number to tell it how far to move.

The turn right Block

This tells the turtle to turn, but it stays in the same spot. You can change the number to tell it how many degrees to turn (90 is a perfect corner, like on a square).

The repeat Block

This is the most powerful block! Any blocks you put inside its C-shape will happen again and again. This is perfect for drawing shapes that have lots of the same sides.

Example Programs to Try

Challenge 1: Draw a Square

This is the best way to use the repeat block. A square has 4 sides and 4 corners, so we repeat the steps 4 times!

  1. Get a repeat block and make sure the number says 4.
  2. Inside it, put a move forward by 100 block.
  3. Under that, still inside the repeat, put a turn right by 90 block.
  4. Click "Run Program" and watch it go!

Challenge 2: Draw a Star! (Hard)

Think you can make a star? The secret is in the angle you turn!

  1. Get a repeat block and set it to 5.
  2. Inside, use move forward by 150.
  3. Then use turn right by 144. (It's a weird number, but it works!)
