Getting Started: Your Digital Studio

The Interface

Pixel Pen is split into two main panels:

  • Block Editor (Left): This is your creative workspace. Drag blocks from the toolbox on the far left to build your programs. The toolbox is organized by category (Pen Actions, Control, etc.) to help you find what you need.
  • Output Panel (Right): This is where you see the results and your text code. You can switch between views using the tabs:
    • Canvas Tab: A blank white canvas where your pen will draw your art.
    • Code Tab: A text editor showing the code that your blocks represent. Advanced users can type code directly here!

Pixel Pen Language Specification

Pixel Pen is a language designed for creating generative art. This is the official reference for all commands.

Pen Actions & Style

draw `distance`

Moves the pen forward. Example: draw 100

turn `degrees`

Rotates the pen clockwise. Use a negative number to turn counter-clockwise. Example: turn -90

lift / drop

Lifts the pen from the canvas (to move without drawing) or drops it back down.

color `color_name`

Changes the pen's color. Example: color red

stroke `width`

Changes the pen's line thickness. Example: stroke 5

Control Flow & Functions

loop `count` ... end_loop

Repeats the code inside the block a specific number of times.

tool `name` ... end_tool

Defines a reusable "tool" (a custom function). Great for organizing complex ideas.

use `name`

Executes a tool you have defined. Example: use draw_a_house

Variables & Memory

set `name` `value`

Creates a "magic number box" (a variable) with a name and stores a value in it. Example: set length 50

change `name` `value`

Adds a value to a variable that already exists. Perfect for making things grow or shrink inside loops. Example: change length 2

Using a Variable

When you want to use the number inside a box, you must write var(name). Example: draw var(length).

Events & Interactivity

when `key_name` ... end_when

Makes your program interactive! The code inside this block only runs when a specific keyboard key is pressed. Valid keys are ArrowUp, ArrowDown, ArrowLeft, ArrowRight, and Space.
